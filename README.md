This project is part of my Data Analytics Internship at **@Algonive**.

- Visualization with Matplotlib & Seaborn

##  Tools
- Python
- Pandas, Seaborn, Matplotlib
- Google Colab

##  Visuals
Includes scatter plots, histograms, and customer segments.
